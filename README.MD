## :information_source: Description

This is a covid-19 tracker dashboard that I coded to practice my react skills, based on an <a href="https://www.youtube.com/watch?v=khJlrj3Y6Ls" target="_blank">youtube tutorial</a>.

  <img width="400" src="https://i.giphy.com/media/eBkaGeJvTuhGMo1c9p/giphy.webp">

## :information_source: Features

- [x] Connection to covid19 public api to retrieve information;
- [x] Multiple charts to shown covid informations;
- [x] Select country to specify covid data information.

## :information_source: How to edit it

To clone this repository and make changes in the source code, you'll need [Git](https://git-scm.com) and  [Nodejs](https://nodejs.org/en/).

```bash
# Clone this repository
$ git clone https://github.com/lucasvtiradentes/covid19-cases-dashboard

# Go into the repository
$ cd covid19-cases-dashboard

# Install dependencies
$ npm install

# Run the code in development mode
$ npm run dev

```
